## 功能簡介
```
1. 每天抓取並監測臺北股市的特定個股(權證)分點進出明細
2. 每天抓取臺北股市的個股價格資料
3. 每天抓取臺北股市的處置股資料
4. 每天抓取臺北股市的股票期貨清單
5. 每周抓取臺北股市的可轉債資料
```

## 使用說明
```
1. 依序安裝好Pyhton環境、必要套件後，到Selenium官網抓取對應瀏覽器版本之chomedrive
2. 將關鍵分點資料輸入StockBranch.csv, 例如:0050,富邦-新竹,台新-高雄
3. 執行主程式stock_branch.py

```
